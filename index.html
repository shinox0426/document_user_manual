Document Tracking System User Manual

Quick, clear guide for daily users and administrators.

1) Login

  1. Open the system URL in your browser.
  2. Enter your username and password.
  3. Click Sign In.
Note: After 5 failed attempts, your account may lock temporarily.

2) Dashboard

  1. Review summary cards for incoming, in-progress, and completed documents.
  2. Check recent activities to see latest updates.
  3. Use shortcuts to open common actions quickly.
Tip: Start each day from the dashboard to spot delayed documents fast.

3) Upload

  1. Select Upload from the menu.
  2. Fill in document details (title, type, owner, date).
  3. Attach the file and click Submit.
Note: Use clear file names like Invoice_2026_05.pdf.

4) Search

  1. Open Search from navigation.
  2. Enter keywords, reference number, or document type.
  3. Apply filters and click Search.
Tip: Combine date and status filters to narrow large result sets.

5) Tracking

  1. Open a document record from search results.
  2. View current status, assigned user, and timeline.
  3. Use tracking history to audit each movement.
Note: Status updates should be done immediately to keep reports accurate.

6) Archiving

  1. Locate completed documents ready for archive.
  2. Set final metadata and retention category.
  3. Click Archive to move records to long-term storage.
Tip: Verify all required approvals before archiving.

7) Roles

  • Admin: Manages users, permissions, and settings.
  • Encoder: Uploads and edits document data.
  • Approver: Reviews and approves workflow steps.
  • Viewer: Can search and view assigned records.
Note: Access is role-based; request changes from your administrator.

8) Troubleshooting

  • Cannot log in: Check password, then use reset link.
  • Upload fails: Confirm file size/type and required fields.
  • Search returns none: Remove filters and retry broader terms.
  • Status looks wrong: Refresh page and review timeline entries.
Tip: When contacting support, include document ID and screenshot.
